# サンレー冷熱株式会社 新ホームページ提案

## コンセプト

燃焼技術で、産業と環境の未来を支える。

既存サイトの信頼感を残しつつ、ファーストビューで技術力と現場感が伝わる構成にします。トップではプロモーションビデオ風の導入を流し、その後に通常の企業サイトとして事業・強み・導入相談へ誘導します。

## トップページ構成

1. 冒頭MV / フルスクリーン
2. ヒーローコピー / CTA
3. 実績訴求
4. 事業領域
5. 技術力・サポート体制
6. MV絵コンテ
7. サイトマップ
8. お問い合わせ導線

## MV 30秒構成案

| 秒数 | 内容 | 狙い |
| --- | --- | --- |
| Combustion | 制御された炎 | 燃焼技術の象徴を一瞬で見せる |
| Equipment | 装置・配管・制御盤 | 産業設備メーカーとしての具体性 |
| Engineering | 図面・組立・検査 | 技術者の手仕事と品質感 |
| Environment | 環境装置・空気感 | 環境対応、省エネ、脱臭の訴求 |
| Support | 点検・メンテナンス | 導入後の安心感 |
| Sunray Reinetsu | ロゴ・コピー | 企業メッセージの定着 |

## 必要素材

- 工場・装置・燃焼シーンの動画または写真
- 製品カテゴリごとの代表写真
- 導入実績の掲載可否
- 会社ロゴデータ
- 会社概要、沿革、採用情報の最新テキスト

## 次工程

1. 写真・動画素材の有無を確認
2. 正式トップページデザインを作成
3. 下層ページの構成を作成
4. 撮影が必要な場合は撮影リストと香盤表を作成
5. 実装または制作会社提出用の仕様書に展開
